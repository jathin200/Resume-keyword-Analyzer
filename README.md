# 📄 Smart Resume Keyword Analyzer

## 📌 Overview
 Resume Keyword Analyzer is a web application that compares a candidate's resume against a job description and calculates a keyword match score. The system helps job seekers optimize resumes for Applicant Tracking Systems (ATS).

## 🚀 Features
- PDF resume upload
- Job description analysis
- ATS keyword matching
- Missing keyword identification
- Match percentage calculation
- Interactive dashboard
- Resume optimization suggestions

## 🛠️ Technologies Used

### Backend
- Python
- Flask

### NLP
- NLTK
- Regex Processing

### Frontend
- HTML
- CSS
- Bootstrap
- JavaScript

### File Processing
- PyPDF2

## ⚙️ Working Process

1. Upload Resume
2. Enter Job Description
3. Extract Keywords
4. Compare Resume and JD
5. Generate ATS Score
6. Display Missing Skills

## 📊 Output
The system provides:
- ATS Match Percentage
- Matched Keywords
- Missing Keywords
- Resume Improvement Suggestions

## 🎯 Benefits
- Improves ATS ranking
- Identifies missing skills
- Helps tailor resumes
- Increases interview chances
